Project Overview

Project Brief

The project envisions a positive future where artificial intelligence (AI) is seamlessly integrated into society. It mostly targets those who worry about their income being significantly impacted by AI advancements, such as artists and creatives. The project aims to alleviate fears surrounding AI by drawing parallels to past technological advancements that initially sparked fear but eventually became integral to daily life.

Target Audience

Artists and creatives, who are currently most affected by AI's encroachment into creative fields, are the primary audience. The project aims to show that AI can be a beneficial and non-threatening tool.

Goals

  1. Demonstrate the peaceful coexistence of AI and human creativity.
  2. Highlight historical technological advancements and their initial fears.
  3. Provide an engaging, interactive platform to explore these themes.

The research inspired the incorporation of interactive features and a user-friendly interface. It reinforced the importance of making complex information accessible and engaging, guiding the design decisions and feature implementations.

Visual Design

Design Development:

The visual design of the project revolves around a futuristic, sleek aesthetic to match the theme of AI integration in 2041. Key design choices include:

  • Color Palette: Dark backgrounds with neon blue accents to evoke a high-tech, futuristic feel.
  • Font Choices: Orbitron, a geometric sans-serif font, to enhance the modern look.

Design Evolution:

Initial designs were simple and focused on functionality. Feedback led to incorporating more visually appealing elements and interactive features. The use of animations and transitions was increased to create a dynamic user experience.

User Interface

Design Patterns:

  1. Tabbed Navigation: Simplifies the interface and reduces cognitive load by organizing content into distinct sections.
  2. Interactive Timeline: Engages users by allowing them to explore historical technological advancements interactively.
  3. Dynamic Content: Elements like the "At this time in history" button provide users with random, engaging facts, enhancing the interactive experience.

Usability Principles:

  • Visibility: Clear, intuitive navigation and interactive elements.
  • Feedback: Visual feedback for user interactions, like button clicks.
  • Consistency: Uniform design elements across all sections.

Example Tasks:

  • Navigating through different tabs (Home, Clock, Search, Article, News, Opinion).
  • Using the search feature to explore information.
  • Interacting with the timeline to learn about historical advancements.

Revisions

User Test Results:

  • Positive Feedback: Users appreciated the clean design and interactive elements.
  • Suggestions for Improvement: Some users found the initial design too static and suggested more dynamic content.

Lessons and Changes:

  • Added more animations and transitions to make the interface feel more dynamic.
  • Improved the interactive timeline based on user feedback to make it more engaging and informative.

Summary

Lessons:

  • Effective UX/UI design requires balancing aesthetics with functionality.
  • User feedback is crucial for iterative improvement.
  • Small interactive elements can significantly enhance user engagement.

Challenges:

Successfully creating an engaging interface that remains interactive while aligning with the project's goal of building a reality and providing accurate context. Which involved balancing the need for factual content with the desire for interactive and engaging elements.
